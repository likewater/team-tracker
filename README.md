# _Hackathon Signup_

#### A Java based application that utilizes Spark, jUnit, Git Hub.

### By Blake Womack

#### User Instructions:

* Add a team
* View teams
* Add a member
* View members by team

### Employs restful routing practices via Spark

### This is the initial release (1.0).
